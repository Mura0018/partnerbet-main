# Hamkor qo'llanmasi — o'z panelingiz bilan ishlash (to'liq)

> Bu hujjat SIZ — hamkor uchun. Panelingizni qanday sozlash va o'z biznesingizni
> qanday yuritishni bosqichma-bosqich tushuntiradi.

---

## 1. Panelga kirish

- Sizga **panel manzili** (`/partner`), **login** va **parol** beriladi.
- Kirasiz — faqat O'ZINGIZGA tegishli ma'lumotni ko'rasiz. Boshqa hamkorlar
  ma'lumotini hech kim ko'rmaydi.
- Ikki rol: **admin** (siz — hammasini boshqarasiz) va **xodim** (faqat
  buyurtmalarni qayta ishlaydi).

---

## 2. Panel bo'limlari (qisqacha)

| Bo'lim | Nima uchun |
|--------|-----------|
| **Boshqaruv (dashboard)** | Buyurtmalar, daromad, holat — bir qarashda |
| **Bot** | O'z Telegram botingizni ulaysiz |
| **Tema** | App ko'rinishini tanlaysiz (bepul/premium) |
| **To'lov usullari** | O'z karta/Click/Payme raqamlaringiz |
| **API** | O'z kompaniyangiz (masalan 1xbet) API'si |
| **Jamoa** | Xodim qo'shish/o'chirish |
| **Hisob** | O'z to'lovlaringiz (obuna/komissiya) |

---

## 3. Botni ulash (eng muhim qadam)

Sizning mijozlaringiz **sizning botingiz** orqali kiradi. Shuning uchun:

1. Telegram'da **@BotFather** ni oching → `/newbot` → bot yarating.
2. BotFather sizga **token** beradi (masalan `12345:AAffff...`).
3. Panel → **Bot** → tokenni qo'ying → **Ulash**.
4. Tizim tekshiradi va **"✅ @sizning_botingiz ulandi"** ko'rsatadi.

> Token **maxfiy** — qayta ko'rsatilmaydi. Xohlagan vaqtda **uzish/almashtirish**
> mumkin. App to'liq sizniki bo'ladi — bizning brend ko'rinmaydi.

---

## 4. Tema tanlash

- **Classic** — bepul, doim mavjud.
- **Neon / Royal** — premium (to'lov asosida ochiladi).
- Panel → **Tema** → tanlang → saqlang. Mijozlaringiz shu ko'rinishда ko'radi.

Premium tema kerak bo'lsa — super admin bilan bog'laning, u ochib beradi.

---

## 5. To'lov usullari

Mijozlar sizga qanday to'lashini o'zingiz belgilaysiz:
- Panel → **To'lov usullari** → Click/Payme/Karta/USDT raqamlaringizni qo'shing.
- Bu raqamlar mijozga to'ldirish paytida ko'rinadi.

---

## 6. API ulash (kassa / 1xbet)

- Panel → **API** → o'z kompaniyangiz API ma'lumotlarini qo'ying.
- Bu buyurtmalarni **avtomatik tekshirish** (o'yinchi ID, balans) uchun ishlatiladi.
- API sirlaringiz **maxfiy** — brauzerga chiqmaydi, faqat server ishlatadi.

---

## 7. Jamoa (xodim qo'shish)

- Panel → **Jamoa** → **Xodim qo'shish** (email + parol).
- Xodim faqat buyurtmalarni qayta ishlaydi — sozlamalarni o'zgartira olmaydi.
- Xodimni istagan vaqtda o'chirasiz.

---

## 8. Ishni boshlash (buyurtmalar)

1. Mijoz botingizga kiradi → hisob to'ldirish/yechish so'raydi.
2. Buyurtma panelingizga tushadi.
3. Siz (yoki xodim) tekshirib **Bajarildi** yoki **Rad etish** qilasiz.
4. Mijoz natijani ko'radi.

---

## 9. Hisob (to'lovlaringiz)

- Panel → **Hisob** — sizning **obuna/komissiya** to'lovlaringiz ro'yxati.
- Holat: **to'lanmagan / to'langan**.
- Vaqtida to'lang — xizmat uzluksiz bo'lishi uchun.

---

## 10. Yordam

- Savol yoki muammo bo'lsa — **Global chat** yoki super admin bilan bog'laning.
- Maqsad — sizda **xatolik yoki nosozlik bo'lmasligi**.

---

## 11. Xavfsizlik (sizning kafolatingiz)

- Ma'lumotlaringiz boshqa hamkorlardan **to'liq ajratilgan**.
- Bot token va API sirlaringiz **hech kimga ko'rinmaydi** (siz ham qayta ko'rmaysiz).
- Faqat siz o'z jamoangiz va sozlamalaringizni boshqarasiz.

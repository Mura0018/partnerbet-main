# Hamkor uchun qo'llanma (onboarding)

Xush kelibsiz! Ushbu qo'llanma siz — hamkor sifatida — platformadan qanday
foydalanishni tushuntiradi. Siz bizning texnologiyamiz bilan **o'z biznesingizni**
yuritasiz: o'z boti, o'z jamoasi, o'z API'lari.

---

## 1. Siz nima olasiz

- **Shaxsiy panel** — faqat sizga tegishli, boshqa hamkorlar ko'rmaydi.
- **O'z Telegram boti** — o'z mijozlaringiz sizning botingiz orqali kiradi.
- **O'z jamoangiz** — admin (siz) + xodimlar.
- **O'z API'laringiz** — masalan 1xbet kassa API'si.
- **Temalar** — 1 ta bepul (doimiy) + 2 ta premium (ixtiyoriy).

---

## 2. Ishga tushish qadamlari

```
1) To'lov / kelishuv
   Super admin siz uchun hamkor profilini ochadi (model: komissiya yoki obuna).

2) Panelga kirish
   Sizga kirish ma'lumotlari beriladi. Panelda faqat O'Z ma'lumotlaringizni
   ko'rasiz.

3) Botni ulash
   BotFather'dan olgan bot tokeningizni panelga kiritasiz.
   Biz uni tekshiramiz va ulaymiz — mijozlaringiz o'sha bot orqali kiradi.
   (Bizning app'dan foydalanmasangiz ham, bot to'liq sizniki bo'ladi.)

4) Tema tanlash
   Bepul "Classic" tema doim mavjud. Premium temalar to'lov asosida ochiladi.

5) Jamoa qo'shish
   Xodimlaringizni qo'shasiz — ular buyurtmalarni qayta ishlaydi.

6) API ulash
   O'z kompaniyangiz (1xbet va h.k.) API ma'lumotlarini kiritasiz.
   API sirlaringiz maxfiy saqlanadi — hech kim (hatto biz ham panelda) ko'rmaydi.

7) Ishni boshlash
   Tayyor! Mijozlaringizga hisob to'ldirish/yechish xizmatini ko'rsatasiz.
```

---

## 3. Bot tokeni haqida

- Tokenni **@BotFather** dan olasiz (`/newbot`).
- Panelda **"Botni ulash"** bo'limiga qo'yasiz.
- Token **maxfiy** — panelda qayta ko'rsatilmaydi, faqat "✅ ulangan" holati ko'rinadi.
- Botni istagan vaqtda **uzish** yoki **almashtirish** mumkin.

---

## 4. Temalar

| Tema | Turi |
|------|------|
| **Classic** | Bepul — doimiy |
| **Neon** | Premium |
| **Royal** | Premium |

Premium tema sizning to'lovingizga qarab super admin tomonidan ochiladi. Temalar
bizning brendimizdan **butunlay farqli** — app to'liq sizniki ko'rinadi.

---

## 5. Xavfsizlik

- Ma'lumotlaringiz **to'liq izolyatsiya** — boshqa hamkorlar ko'rmaydi.
- API sirlaringiz va bot tokeningiz **brauzerga chiqmaydi** (faqat server ishlatadi).
- Faqat siz (partner admin) o'z jamoangizni va sozlamalaringizni boshqarasiz.

---

## 6. Yordam

Savol yoki muammo bo'lsa — **Global chat** yoki super admin bilan bog'laning.
Maqsadimiz — sizda **xatolik yoki nosozlik bo'lmasligi**.
